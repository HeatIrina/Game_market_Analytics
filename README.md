# Аналитика игрового рынка

## Описание проекта

Разработать интерактивный дашборд в Apache Superset для комплексного анализа игрового рынка за период 2023–2025 годов. Цель — обеспечить продуктовых менеджеров, аналитиков и маркетологов единой точкой контроля метрик для оптимизации портфеля игр, повышения вовлеченности игроков и корректировки ценовой стратегии.

## Этапы проекта

1. Анализ требований и проектирование: определение целевой аудитории и ключевых бизнес-целей: отслеживание воронки от установки игр до получения достижений, анализ ценовой динамики.
2. Подготовка данных и расчёт метрик: объединение данных из связанных таблиц (`games`, `prices`, `players`, `players_games`, `achievements, history`). Расчёт ключевых метрик.
3. Построение визуализаций: создание четырёх вкладок — «Игровые достижения», «Анализ игроков», «Стоимость игр» и «Анализ игр».
4. Настройка интерактивности: добавление глобальных фильтров и локального фильтра для YoY анализа. Настройка кросс-фильтрации между всеми визуализациями.
5. Формирование аналитической записки: выявление положительных тенденций и областей для улучшения. Разработка трёх приоритетных рекомендаций для продуктового отдела.

## Навыки и инструменты

**Навыки**
1. Проектирование дашборда.
2. Визуализация данных.
3. Настройка фильтров и кросс-фильтрации.
4. Расчёт метрик.
5. Когортный анализ.
6. Работа с 6 связанными таблицами.
7. Обработка NULL.
8. Формирование аналитических выводов и рекомендаций.

**Инструменты**
`Apache Superset`, `SQL`.

## Вывод
Разработанный дашборд обеспечил комплексный мониторинг игрового рынка по трём ключевым направлениям: вовлеченность игроков (отслеживание достижений, доли редких достижений по жанрам), пользовательская активность (распределение игроков по странам и платформам, топ-жанры по установкам, YoY динамика) и ценовая политика (средние и медианные цены по жанрам, динамика цен по платформам, выявление игр с высокой волатильностью).

На основе данных дашборда сформулированы три приоритетные рекомендации для департамента продуктового менеджмента: оптимизация портфеля игр (увеличение количества игр в жанрах с высокой вовлеченностью, диверсификация платформенного покрытия, балансировка портфеля по ценовым сегментам), улучшение вовлеченности игроков (разработка большего числа редких достижений для популярных жанров, оптимизация прогрессии достижений, внедрение системы мотивации) и корректировка ценовой стратегии (стабилизация цен на волатильные игры, внедрение динамического ценообразования, оптимизация скидочных кампаний).

Интерактивные фильтры и кросс-фильтрация позволяют пользователям самостоятельно исследовать причины изменений ключевых метрик, а цветовое кодирование направлений изменения обеспечивает мгновенное восприятие динамики роста и падения показателей.
